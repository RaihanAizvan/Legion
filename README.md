# Legion - Minecraft Server Website

Legion is a website for the **Legion Minecraft Server** built using:
- Express.js
- MongoDB
- EJS
- Tailwind CSS

This project follows **MVC Architecture** and uses **ES Modules** (`type: "module"`).
